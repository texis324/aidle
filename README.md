# AIDLE — 純生産性を奪い合え

**人間がAIを使いまくるゲーム。** あなたは何百ものAIを回す指揮官。だが大群は直接見えない —— 上がってくるのは **司令塔（総統AI）の「報告」** だけ。

▶ **遊ぶ:** https://texis324.github.io/aidle/ （クリック／タップだけ・スマホ可）

## 遊び方
- 各報告カードを即断 → **出荷**（信じて即出荷・速いがスロップ混入リスク）／ **検証**（集中を消費して精査・スロップ除去）／ **捨てる**
- **⚠過負荷** の司令塔は確信度を盛って嘘をつく。鵜呑み注意。
- 報告を溜めると **焦燥** が満ちて燃え尽き。スロップを出荷しすぎると **信頼** が尽きてクビ。
- ¥（純生産性）で AI や司令塔を増やせる。**増やすほど儲かるが、増やすほど溺れる。**
- 数だけの **並列農場** に、純生産性で勝て。

## 設計メモ
「AIを止めなければ勝ち」ではない（並列で回すだけの相手に負ける）。本当のボトルネックは **人間の判断**。AIを増やしても判断帯域は増えない —— だから増やすほど溺れる。

司令塔を増やして群を分担すれば過負荷（＝嘘）は消えるが、報告の本数が増える。フラットすぎても、階層を盛りすぎても詰む —— **組織図そのものがゲーム**。
